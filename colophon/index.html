Colophon

This site is built using Eleventy (11ty). I’ve tried a lot of web frameworks but I always end up back with 11ty. It’s simple, flexible, has a lot of useful plugins, and just gets the job done. The site is hosted on GitHub Pages.

Fonts used are based on a slightly modified Modern Font Stacks. Heading text uses transitional typefaces and body text uses neo-grotesques. Code blocks are set in IBM Plex Mono.

The code for this site is open source under the MIT license and is available on GitHub. Content on this site, such as articles and snippets, is licensed under the Creative Commons license CC BY-NC-SA 4.0.

This site doesn’t use cookies or analytics scripts.
